# IndividualSkillsMarketplace — Resumen conciso

Descripción
- Mercado para compra/activación de skills individuales sin minting NFT asociado. Gestiona activaciones, expiraciones, y notifica al staking/leveling cuando corresponde.

Usuario
- Comprar y activar skills
- Desactivar y recibir reembolsos si aplica

Admin
- Cargar/editar catálogo de skills
- Configurar contratos de staking y precios

Interacciones
- Staking: notificaciones sobre activación/desactivación
- Core Marketplace: enlace para gestionar skills vinculados a NFTs

- `contracts/Marketplace/IndividualSkillsMarketplace.sol`
